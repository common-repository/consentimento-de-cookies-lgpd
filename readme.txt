== Description ==

= O que é LGPD? =

A Lei Geral de Proteção de Dados (LGPD) é uma lei brasileira  destinada a  fortalecer e unificar a proteção de dados pessoais dos cidadãos do Brasil.
Para saber mais, acesse [https://privally.io/](https://privally.io/).

= LGPD e Consentimento de Cookies =

A LGDP afirma que, como proprietário de um site, você não pode presumir que um usuário tenha optado pelos cookies usados ​​em seu site, o que chamamos de consentimento implícito. O usuário deve realizar uma opção positiva, explícita, uma  “ação afirmativa” para sinalizar seu consentimento para o uso de cookies e os mesmos não pode ser forçado  a optar pelo uso de cookies.

Na prática, não é possível sequer rastrear nossos usuários com o Google Analytics sem que os mesmos forneçam seu consentimento consciente, informados por você sobre o que cada Cookie faz.

= Como este plugin funciona =

Este plug-in foi projetado para ajudar você a preparar seu website para as  exigências da LGPD e GDPR relacionadas a cookies.
Você deve cadastrar seu site através do [Portal Privally](https://privally.io/portal/), para que a Privally classifique os cookies de seu site automaticamente e forneça a você um código de cliente Privally gratuitamente.
Para configurar seu Código de Cliente no Plugin de Consentimento de Cookies LGPD Privally, acesse a opção “Geral > Consentimento de Cookies LGPD” e informe seu Código de Cliente.

== Installation ==

1. Faça o Upload do plugin na pasta '/wp-content/plugins/'.
1. Ative o Plugin
1. Entre no site [https://privally.io/portal](https://privally.io/portal) cadastre a url do seu site e gere seu código de cliente.
1. Informe seu código de cliente nas configurações do plugin.
1. Crie uma p&aacute;gina com o título de "Política de Privacidade" e inclua o shortcode [privacy-policy] no corpo da página.

Após instalação a sua barra de consentimento de cookies estará disponível para seus visitantes em seu site, assim como a política de privacidade.
Você pode configurar o layout da barra de cookies e o texto da política de privacidade no portal.
